# HEAD-QA
## 📦 Dataset Overview
HEAD-QA is a dataset for evaluating the quality of digital human head editing. Using 150 prompt words, 1655 edited videos were constructed from avatars in the NeRsemble dataset using 6 different methods.

## 🔍 Dataset Details
The HEAD-QA dataset consists of three parts.
- **Edited_Videos:**  1655 edited head videos
- **Origin_Videos:**  22 origin head videos
- **Prompt.json:**    150 text prompts
